⚙️ הגדרה חד-פעמית נדרשת – Firebase (חינמי, 5 דקות)

  1. היכנס ל-console.firebase.google.com עם חשבון Google
  2. לחץ Create a project → תן שם → המשך
  3. בתפריט שמאל: Build → Firestore Database → Create database → בחר Start in test mode → Next → Enable
  4. בתפריט שמאל: Project settings (גלגל שיניים) → Your apps → </> Web → רשום אפליקציה
  5. העתק את האובייקט firebaseConfig שמקבל ופתח את ה-HTML בעורך טקסט
  6. מצא את const firebaseConfig = { ... } בראש הקובץ והחלף בנתונים שלך
  7. שמור ופתח מחדש – הסנכרון יעבוד מכל מכשיר ומכל מקום! 🎉
